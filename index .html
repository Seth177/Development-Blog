Positioning

  1. Relative
    • Moves elements around relative to
      where they would appear on the page
  2. Absolute
    • Offset is relative to the entire browser window instead of the original position of the element.

    • Absolute position elements are placed in relation to a parent content(browser viewport, or parent element). The parent is the positioning context

    • If the parent element has the position of relative and the child element has the position of absolute, the child will move according to the dimensions of the parent

  3. Fixed
    • Fixed elements don't scroll with the rest of the page

  4. Z-Index
    • Z-index property lets you control the depth of elements on the page

    • Negative z-index go farther into the page. (behind)

    • Positive z-index comes out of the page

Week 4

Responsive images

  • It is important to set the width of pictures to percentages instead of pixels
    • Pixels makes the image have a fix size
    • Percentages makes the image have the same size while adjusting the screen
  • Max-width:The max-width property sets a maximum width for an element, which does not allow the width of that element to be larger than its max-width value (but it can be smaller).
  • Use background-size property to make image adjust to the background
  • flex-basis -> The flex-basis property specifies the initial length of a flexible item.
  • flex-grow -> The flex-grow property specifies how much the item will grow relative to the rest of the flexible items inside the same container.
  • flex-grow: inherit -> Ensure the flex-grow property is applied
