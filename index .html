Media Queries

Positioning

  1. Relative
    • Moves elements around relative to
      where they would appear on the page
  2. Absolute
    • Offset is relative to the entire browser window instead of the original position of the element.

    • Absolute position elements are placed in relation to a parent content(browser viewport, or parent element). The parent is the positioning context

    • If the parent element has the position of relative and the child element has the position of absolute, the child will move according to the dimensions of the parent

  3. Fixed
    • Fixed elements don't scroll with the rest of the page

  4. Z-Index
    • Z-index property lets you control the depth of elements on the page

    • Negative z-index go farther into the page. (behind)

    • Positive z-index comes out of the page
